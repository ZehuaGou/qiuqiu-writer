# SmartReads 单章处理逻辑更新说明

## 📋 更新概述

已将拆书分析功能更新为使用 **SmartReads 的单章处理逻辑**，每一章单独进行 AI 分析，而不是按章节组批量处理。

## 🔄 主要变更

### 1. **章节处理方式改变**

**之前**：
- 将章节按组大小（10/20/50/100/200章）聚合成组
- 一次分析一组章节（包含多章内容）
- 结果按组展示

**现在**：
- 每一章单独识别和处理
- 每一章单独调用 AI 分析
- 结果按单章展示
- **完全遵循 SmartReads 的处理逻辑**

### 2. **章节识别逻辑**

采用 SmartReads 的章节识别正则表达式：

```typescript
const headingPattern = /(^\s*(?:第\s*[0-9一二三四五六七八九十百千零]+\s*[章回节卷篇]|(?:Chapter|CHAPTER)\s*\d+|序章|楔子|尾声|后记|番外)[^\n]*\n)/gm;
```

**支持的章节格式**：
- `第1章`、`第一章`、`第五十章` 等中文格式
- `Chapter 1`、`CHAPTER 10` 等英文格式
- `序章`、`楔子`、`尾声`、`后记`、`番外` 等特殊章节

### 3. **数据结构变更**

```typescript
// 之前：ChapterGroup
interface ChapterGroup {
  name: string;        // 如：第1-50章
  content: string;     // 多章内容合并
}

// 现在：Chapter
interface Chapter {
  id: string;          // 唯一标识
  title: string;       // 章节标题
  content: string;     // 单章内容
  number: number;      // 章节号
}
```

### 4. **分析流程变更**

**之前的流程**：
```
上传文件 → 按组拆分 → 选择组 → 批量分析组
```

**现在的流程**（SmartReads 方式）：
```
上传文件 → 识别章节 → 选择章节 → 逐章分析
```

## ✨ 新功能特点

### 1. **精确的章节识别**

- 只识别行首的章节标题
- 自动过滤无效/过短内容
- 自动提取章节号（支持中文数字转阿拉伯数字）

### 2. **单章独立分析**

- 每章使用完整的 SmartReads 提示词
- 每章生成独立的分析结果
- 更精确的章节级分析

### 3. **更好的UI体验**

- 显示章节号标签
- 实时显示当前分析的章节
- 进度百分比显示
- 按章节选择分析

### 4. **智能默认选择**

- 识别完成后，默认选择前 5 章
- 方便快速开始分析

## 🎯 使用方法

### 1. 上传小说文件

选择一个 `.txt` 格式的小说文件。

### 2. 识别章节

点击 **"识别章节"** 按钮，系统会：
- 自动识别所有章节标题
- 提取章节号和标题
- 计算每章字符数
- 显示章节列表

### 3. 选择章节

- 从列表中勾选要分析的章节
- 可以使用 **"全选"** 或 **"取消全选"**
- 支持多选（建议一次不超过 20 章）

### 4. 开始分析

点击 **"开始 AI 分析"** 按钮：
- 按选择顺序逐章分析
- 实时显示分析进度
- 流式输出分析内容
- 每章间隔 1 秒避免 API 限流

### 5. 查看结果

- 右侧面板实时显示每章的分析结果
- 包含完整的 8 列分析表格
- 支持下载为 Markdown 文件

## 📊 分析输出格式

每一章的分析结果包含以下内容（符合 SmartReads 规范）：

| 栏目 | 说明 |
|------|------|
| **章节号** | 阿拉伯数字格式的章节号 |
| **章节标题** | 原始章节标题 |
| **章节核心剧情梗概** | 2-3句精炼概括 |
| **本章核心功能/目的** | 分析战略意义 |
| **画面感/镜头序列** | JSON数组格式，3-5个关键画面 |
| **关键情节点** | JSON数组格式，关键节点 |
| **本章氛围/情绪** | JSON数组格式，情感体验 |
| **结尾"钩子"** | 悬念设置 |

## 🔧 技术实现

### 章节拆分逻辑

参考 SmartReads 的 `useFileHandler.js` 中的 `splitTextToChapters` 函数：

```typescript
// 1. 使用正则表达式按章节标题分割
const parts = text.split(headingPattern);

// 2. 提取标题和内容
for (let i = 1; i < parts.length; i += 2) {
  const title = parts[i].trim();
  const content = parts[i + 1].trim();
  // 过滤短内容
  if (content.length > 50) {
    chapters.push({ title, content });
  }
}
```

### 章节号提取

```typescript
// 支持中文数字转阿拉伯数字
convertChineseNumberToArabic("五十一") // => 51
```

### 逐章分析

参考 SmartReads 的 `useAnalyzer.js` 中的 `analyzeMultipleFiles` 函数：

```typescript
for (let i = 0; i < chapters.length; i++) {
  // 单章分析
  const result = await analyzeChapterContent(
    chapter.content,
    progressCallback,
    settings
  );
  
  // 章节间延迟 1 秒
  if (i < chapters.length - 1) {
    await new Promise(resolve => setTimeout(resolve, 1000));
  }
}
```
