# 模型元数据展示功能说明

## 🎯 功能概述

现在每章的分析结果都会显示AI模型的返回信息（元数据），包括使用的模型、参数配置、耗时、Token消耗等详细信息。

## 📊 显示的元数据信息

### 1. **🤖 模型名称**
- 显示使用的AI模型
- 示例：`codedrive-chat`、`gpt-4`、`gpt-3.5-turbo`

### 2. **🌡️ 温度参数**
- 模型的temperature设置
- 控制输出的随机性
- 范围：0.0 - 1.0
- 示例：`0.7`

### 3. **⏱️ 分析耗时**
- 从开始到完成的实际耗时
- 单位：秒
- 示例：`23.45秒`

### 4. **📊 Token使用量**
- 估算的Token消耗
- 用于成本估算
- 示例：`3520 tokens`

## 🔧 技术实现

### 后端修改

#### 1. ai_service.py - 添加元数据返回

**开始消息**：
```python
start_msg = json.dumps({
    "type": "start", 
    "message": "开始分析章节内容...",
    "metadata": {
        "model": model,
        "temperature": temperature,
        "max_tokens": max_tokens,
        "start_time": start_time.isoformat()
    }
})
```

**完成消息**：
```python
done_msg = json.dumps({
    "type": "done", 
    "message": "分析完成",
    "metadata": {
        "model": model,
        "temperature": temperature,
        "max_tokens": max_tokens,
        "duration": round(duration, 2),
        "estimated_tokens": completion_tokens,
        "end_time": end_time.isoformat()
    }
})
```

### 前端修改

#### 1. 数据结构更新

**AnalysisResult接口**：
```typescript
interface AnalysisResult {
  fileName: string;
  content: string;
  isComplete: boolean;
  hasError: boolean;
  timestamp: number;
  metadata?: {
    model?: string;
    temperature?: number;
    max_tokens?: number;
    duration?: number;
    estimated_tokens?: number;
    start_time?: string;
    end_time?: string;
  };
}
```

#### 2. 元数据采集

在分析过程中，通过进度回调收集元数据：

```typescript
let metadata: any = {};

const analysisResult = await analyzeChapterContent(
  chapter.content,
  (progress) => {
    // 保存元数据
    if (progress.metadata) {
      metadata = { ...metadata, ...progress.metadata };
    }
  },
  settings
);

// 保存到结果
setAnalysisResults(prev => ({
  ...prev,
  [chapter.id]: {
    ...result,
    metadata: metadata
  }
}));
```

#### 3. UI展示组件

```tsx
{result.metadata && (

```
