## 🚀 使用指南

### 1. 环境变量配置

在 `frontend/` 目录下创建 `.env` 文件（或 `.env.local`）：

```bash
# frontend/.env
VITE_API_URL=http://localhost:8001
```

### 2. 启动后端服务

```bash
# 方式1: 使用 product_api（包含作品和章节接口）
cd backend
python3 -m memos.api.product_api --port 8001

# 方式2: 使用 ai_api（包含所有接口）
python3 -m memos.api.ai_api --port 8001
```

### 3. 启动前端服务

```bash
cd frontend
npm install
npm run dev
```

## ⚠️ 注意事项

### 1. 认证
- 所有需要认证的接口都会自动从 `localStorage` 获取 `access_token`
- 如果 token 过期，需要调用 `authApi.refreshToken()` 刷新
- 401 错误时应该清除 token 并跳转到登录页

### 2. CORS
- 开发环境允许所有来源（`*`）
- 生产环境需要修改 `product_api.py` 和 `ai_api.py` 中的 CORS 配置，限制具体域名

### 3. 错误处理
- 所有 API 客户端都包含错误处理
- 错误信息会显示在控制台和用户界面
- 建议添加全局错误处理中间件

### 4. 类型安全
- 所有 API 客户端都使用 TypeScript 类型定义
- 接口返回的数据类型与后端 Schema 对应
