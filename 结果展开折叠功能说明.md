# 分析结果展开/折叠功能说明

## 🎯 功能概述

分析结果现在支持展开/折叠功能，解决了多个章节分析结果同时显示导致页面过长、新结果覆盖旧结果的问题。

## ✨ 主要特性

### 1. **点击展开/折叠**
- 点击章节标题栏即可展开或折叠该章节的结果
- 展开时显示完整的分析内容和元数据
- 折叠时只显示标题和基本信息

### 2. **自动展开当前分析**
- 正在分析的章节会自动展开
- 方便实时查看分析进度
- 分析完成后保持展开状态

### 3. **批量操作**
- **展开全部**：一键展开所有章节结果
- **折叠全部**：一键折叠所有章节结果
- 方便快速浏览或整理

### 4. **状态标识**
- **完成**：绿色标签（分析成功）
- **失败**：红色标签（分析失败）
- **分析中...**：黄色标签（正在分析）
- **耗时**：显示在标题栏右侧

## 🔧 技术实现

### 1. 状态管理

```typescript
const [expandedChapters, setExpandedChapters] = useState<Set<string>>(new Set());
```

使用 `Set` 数据结构存储已展开的章节ID，高效且易用。

### 2. 展开/折叠函数

```typescript
// 切换单个章节
const toggleChapterExpand = (chapterId: string) => {
  setExpandedChapters(prev => {
    const newSet = new Set(prev);
    if (newSet.has(chapterId)) {
      newSet.delete(chapterId);
    } else {
      newSet.add(chapterId);
    }
    return newSet;
  });
};

// 展开全部
const expandAll = () => {
  setExpandedChapters(new Set(Object.keys(analysisResults)));
};

// 折叠全部
const collapseAll = () => {
  setExpandedChapters(new Set());
};
```

### 3. 自动展开逻辑

在开始分析时自动展开：

```typescript
// 自动展开当前分析的章节
setExpandedChapters(prev => new Set([...prev, chapter.id]));
```

### 4. 条件渲染

```typescript
{isExpanded && (
  <div className="result-body">
    {/* 元数据和内容 */}
  </div>
)}
```

## 📋 使用场景

### 场景1：分析多个章节

**问题**：分析50章，所有结果同时显示，页面很长

**解决**：
1. 默认只展开正在分析的章节
2. 已完成的章节自动折叠
3. 需要查看时点击展开
4. 使用"展开全部"快速查看所有结果

### 场景2：对比特定章节

**操作**：
1. 折叠所有章节
2. 只展开需要对比的2-3章
3. 方便并排查看和对比

### 场景3：快速浏览

**操作**：
1. 所有章节折叠
2. 只看标题和状态
3. 快速找到需要查看的章节
4. 点击展开查看详情
