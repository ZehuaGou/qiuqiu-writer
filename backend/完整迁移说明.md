# 完整API迁移说明

## 📋 迁移概述

已将 `backend/app/` 下的所有API相关文件迁移到 `backend/src/memos/api/` 目录下，实现统一的API管理。

## ✅ 已完成的工作

### 1. 目录结构迁移

**迁移的目录和文件**：

#### Core模块 (`app/core/` → `src/memos/api/core/`)
- `config.py` - 应用配置
- `database.py` - 数据库连接和会话管理
- `redis.py` - Redis连接管理
- `security.py` - 安全认证和权限管理

#### Models模块 (`app/models/` → `src/memos/api/models/`)
- `user.py` - 用户模型
- `work.py` - 作品模型
- `chapter.py` - 章节模型
- `template.py` - 模板模型
- `characters.py` - 角色模型
- `writing.py` - 写作相关模型
- `system.py` - 系统模型

#### Schemas模块 (`app/schemas/` → `src/memos/api/schemas/`)
- `auth.py` - 认证相关的Pydantic模式
- `work.py` - 作品相关的Pydantic模式

#### Routers模块 (已迁移)
- `auth_router.py` - 认证路由
- `chapters_router.py` - 章节路由
- `templates_router.py` - 模板路由
- `works_router.py` - 作品路由

#### Services模块 (已迁移)
- `auth_service.py` - 认证服务
- `chapter_service.py` - 章节服务
- `sharedb_service.py` - ShareDB服务
- `template_service.py` - 模板服务
- `user_service.py` - 用户服务
- `work_service.py` - 作品服务

### 2. 导入路径更新

所有导入路径已从 `app.*` 更新为 `memos.api.*`：

#### Core模块导入
```python
# 之前
from app.core.config import get_settings
from app.core.database import get_async_db
from app.core.redis import get_redis
from app.core.security import get_current_user_id

# 之后
from memos.api.core.config import get_settings
from memos.api.core.database import get_async_db
from memos.api.core.redis import get_redis
from memos.api.core.security import get_current_user_id
```

#### Models模块导入
```python
# 之前
from app.models.work import Work, WorkCollaborator
from app.models.chapter import Chapter, ChapterVersion

# 之后
from memos.api.models.work import Work, WorkCollaborator
from memos.api.models.chapter import Chapter, ChapterVersion
```

#### Schemas模块导入
```python
# 之前
from app.schemas.work import WorkCreate, WorkResponse
from app.schemas.auth import LoginRequest, RegisterRequest

# 之后
from memos.api.schemas.work import WorkCreate, WorkResponse
from memos.api.schemas.auth import LoginRequest, RegisterRequest
```

#### Services模块导入
```python
# 之前
from app.services.work_service import WorkService
from app.services.chapter_service import ChapterService

# 之后
from memos.api.services.work_service import WorkService
from memos.api.services.chapter_service import ChapterService
```

### 3. 配置文件更新

#### `core/__init__.py`

#### `models/__init__.py`

#### `app/main.py`

## 🚀 使用方式

### 启动服务

#### 方式1：使用统一API服务（推荐）
```bash
python3 -m memos.api.ai_api --port 8001
```

这将启动包含所有路由的统一API服务。

#### 方式2：使用原有主应用
```bash
python3 -m app.main
```

### 访问API

所有接口现在都在统一的API服务下：

- **WriterAI应用接口** (`/api/v1/*`)
  - `/api/v1/auth/*` - 认证接口
  - `/api/v1/chapters/*` - 章节接口
  - `/api/v1/templates/*` - 模板接口
  - `/api/v1/works/*` - 作品接口

- **AI分析接口** (`/ai/*`)
  - `/ai/health` - 健康检查
  - `/ai/analyze-chapter` - 章节分析
  - `/ai/default-prompt` - 默认提示词

- **产品接口** (`/product/*`)
  - 各种产品API
